# Proyecto Streamlit - Análisis de Vehículos

Esta aplicación web fue creada con Streamlit y permite analizar un conjunto de datos de anuncios de venta de coches en Estados Unidos.

### Funcionalidades:
- Visualización de un histograma de kilometraje (`odometer`)
- Visualización de un gráfico de dispersión (`price` vs `odometer`)

### Ejecución:
Para ejecutar la aplicación localmente:
```bash
streamlit run app.py
