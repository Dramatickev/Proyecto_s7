# Proyecto Streamlit - Análisis de Vehículos

Dashboard interactivo en Streamlit para explorar datos de anuncios de vehículos y visualizar patrones con Plotly.

## Link de la aplicación web
https://proyecto-s7-7pnk.onrender.com

### Descripción del proyecto:
Este proyecto forma parte del Sprint 7 y tiene como objetivo aplicar herramientas de desarrollo de software mediante la creación de una aplicación web interactiva desarrollada con Streamlit.

La aplicación permite visualizar y analizar un conjunto de datos de anuncios de venta de coches en Estados Unidos, utilizando las bibliotecas Pandas, Plotly y Streamlit. A través de un panel de control sencillo e interactivo, el usuario puede explorar las características del dataset y generar visualizaciones dinámicas que facilitan el análisis exploratorio de los datos.

### Funcionalidades principales:
- Visualización de histogramas: permite crear un histograma interactivo que muestra la distribución del kilometraje (odometer) de los vehículos.

- Gráfico de dispersión: muestra la relación entre el precio (price) y el kilometraje (odometer), ayudando a identificar patrones o tendencias en los datos.

- Interactividad: los gráficos se generan dinámicamente mediante botones o casillas de verificación, lo que permite al usuario decidir qué tipo de visualización desea ver.

- Implementación web: la aplicación puede ejecutarse localmente o desplegarse en la nube (por ejemplo, mediante Render), haciendo accesible el análisis desde cualquier navegador.

## Propósito del proyecto:
El propósito principal es practicar el flujo completo de trabajo en ingeniería de software y análisis de datos:

- Crear y gestionar entornos virtuales de Python.
- Desarrollar un panel de control web con Streamlit.
- Integrar visualizaciones interactivas basadas en Plotly.
- Publicar y mantener el proyecto en GitHub.
